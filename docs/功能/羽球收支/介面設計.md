# 羽球收支管理系統 介面設計

## 設計方向

### 美學定位

**風格**：專業簡潔 × 運動活力

結合財務系統的專業信任感與羽球運動的動感元素。避免過於企業化的冰冷感，保留社團的親切氛圍。

### 設計原則

| 原則 | 說明 |
|------|------|
| 清晰優先 | 數據表格易讀，金額醒目 |
| 操作簡單 | 主要功能一目瞭然 |
| 響應式 | 手機、平板、桌機皆可用 |
| 一致性 | 統一的顏色與間距系統 |

## 元件設計

### 統計卡片

```html
<div class="bg-white rounded-xl shadow-sm border border-slate-200 p-6">
  <p class="text-sm font-medium text-slate-500">總收入</p>
  <p class="text-3xl font-bold text-emerald-600 mt-2">$12,500</p>
</div>
```

**規格**：
- 圓角：`rounded-xl` (12px)
- 陰影：`shadow-sm`
- 邊框：`border border-slate-200`
- 內距：`p-6` (24px)

### 按鈕樣式

**主要按鈕**
```html

               px-4 py-2 rounded-lg transition-colors">

```

```html
               px-4 py-2 rounded-lg border border-slate-300 transition-colors">
  匯出 CSV
</button>
```

**圖示按鈕**

## 互動細節

### Hover 效果

- 按鈕：背景色加深、`transition-colors`
- 表格列：`hover:bg-slate-50`
- 圖示按鈕：顏色變化 + 背景圓形

### Focus 狀態

- 輸入框：`focus:ring-2 focus:ring-blue-500`
- 按鈕：`focus:outline-none focus:ring-2 focus:ring-blue-500 focus:ring-offset-2`

### 動畫

| 元素 | 動畫 |
|------|------|
| Dialog 開啟 | fade in + scale up |
| Toast 出現 | slide in from right |
| 資料載入 | skeleton pulse |
| 刪除確認 | shake |
